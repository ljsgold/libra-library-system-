# 分类图片资源说明

本目录用于存放图书分类的展示图片。

## 分类体系

本系统采用**中国图书馆分类法（中图法）22大类**标准分类体系，符合顶级公共图书馆的分类标准。

## 图片要求

- **尺寸建议**: 400x300px 或更高分辨率（16:9 或 4:3 比例）
- **格式**: JPG 或 PNG
- **文件大小**: 建议控制在 200KB 以内
- **风格**: 现代、专业、符合顶级图书馆氛围，避免学校教务风格

## 当前实现

目前系统使用 **Unsplash** 的专业图书馆风格图片，通过 CDN 直接引用。如需替换为本地图片，请下载对应图片并放置在此目录下。

## 中图法22大类分类

1. **A** - 马克思主义、列宁主义、毛泽东思想、邓小平理论
2. **B** - 哲学、宗教
3. **C** - 社会科学总论
4. **D** - 政治、法律
5. **E** - 军事
6. **F** - 经济
7. **G** - 文化、科学、教育、体育
8. **H** - 语言、文字
9. **I** - 文学
10. **J** - 艺术
11. **K** - 历史、地理
12. **N** - 自然科学总论
13. **O** - 数理科学和化学
14. **P** - 天文学、地球科学
15. **Q** - 生物科学
16. **R** - 医药、卫生
17. **S** - 农业科学
18. **T** - 工业技术
19. **U** - 交通运输
20. **V** - 航空、航天
21. **X** - 环境科学、安全科学
22. **Z** - 综合性图书

## 图片资源推荐

### 免费高质量图片库

- [Unsplash](https://unsplash.com/) - 高质量免费图片（当前使用）
- [Pexels](https://www.pexels.com/) - 免费图片和视频
- [Pixabay](https://pixabay.com/) - 免费图片、插图和矢量图

### 搜索关键词建议

- `library books`, `bookshelf`, `reading room`, `modern library`
- `professional library`, `public library`, `book collection`
- 避免：`school library`, `university library`, `campus`（避免学校教务风格）

## 注意事项

- 如果某个分类图片缺失，系统会自动使用 SVG 占位图
- 图片路径配置在 `Home.vue` 的 `categoryConfig` 中
- 建议使用有版权的免费图片，避免侵权问题
- 如需使用本地图片，请将图片文件放置在此目录下，并更新 `Home.vue` 中的图片路径
